# NewsMind Frontend

Next.js frontend for the NewsMind Humas DPRD platform.

## Scope

- Login
- Dashboard statistik media
- Kliping berita
- Upload PDF / link input
- Approval workflow
- Arsip berita
- AI outputs: summary, review, sentiment, topic, caption, web draft

## Setup

```bash
bun install
bun run dev
```

## Environment

```env
NEXT_PUBLIC_API_URL=http://localhost:4000/api/v1
```

## Notes

- Target deployment: Vercel
- Keep UI focused on newsroom and Humas workflows
