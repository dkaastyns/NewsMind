# NewsMind

Sistem Cerdas Humas DPRD Berbasis AI.

NewsMind adalah monorepo untuk platform kliping berita digital dan workflow
persetujuan berbasis AI. Fokus utamanya adalah membantu tim Humas DPRD
mengelola berita dari link atau upload PDF, lalu menghasilkan ringkasan,
ulasan, analisis sentimen, klasifikasi topik, caption media sosial, dan draft
berita website.

## V1 Scope

- Kliping berita digital
- Input berita dari link atau upload PDF
- AI summary, ulasan, sentimen, dan klasifikasi topik
- Dashboard statistik media
- Arsip berita
- Generate caption media sosial
- Generate berita website
- Approval workflow
- Manajemen pengguna

## Repository Layout

- `frontend/` - Next.js app
- `backend/` - NestJS API
- `ai-service/` - FastAPI AI service
